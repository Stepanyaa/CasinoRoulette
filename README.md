| Command                     | Description                              | Permission                   |
|-----------------------------|------------------------------------------|------------------------------|
| `/casino`                   | Open the main casino menu                | `casino.admin` (for full access) |
| `/casino reload`            | Reload configuration and messages        | `casino.reload`             |

## 🔧 **Permissions**

- **`casino.admin`** – Full administrative access (includes all features)
  - Default: OP
  - Children permissions:
    - `casino.reload` – Reload plugin configuration
    - `casino.update.notify` – Receive update notifications
    - `casino.limit.admin` – Maximum betting position limit (25)

- **`casino.reload`** – Reload plugin configuration
  - Default: OP

- **`casino.limit.vip`** – Increased betting position limit (10)
  - Default: false

- **`casino.limit.admin`** – Maximum betting position limit (25)
  - Default: false

- **`casino.update.notify`** – Receive update notifications
  - Default: OP
